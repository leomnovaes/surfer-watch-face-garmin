Shore Watch — a surfer-focused watch face for the Garmin Instinct 2X Solar.

Tide times, weather, wind, moon phase, heart rate, stress, and more — all on one screen, no menus.

WORKS OUT OF THE BOX — no API keys required for weather. Uses Garmin's built-in weather data by default.

FEATURES
• Next tide time, direction (high/low), and predicted height via StormGlass API
• Weather condition with day/night icons and temperature
• Wind direction arrow (rotated to exact degree) and speed (configurable: km/h, knots, mph, m/s)
• Precipitation chance
• Sunrise/sunset with next event time
• Moon phase (28 phases, computed locally)
• Heart rate with stress arc gauge
• Battery, notifications, Bluetooth connectivity
• Custom clock font (switchable via settings)
• Seconds on wrist raise

WEATHER SOURCES
• Default: Garmin built-in weather (zero config, updates via phone)
• Optional: OpenWeatherMap 2.5 (free, no credit card, updates every 5 min)

SETUP
• Weather works immediately — no setup needed
• For tide data: get a free StormGlass API key at stormglass.io
• For OWM weather: get a free key at openweathermap.org (optional)
• Enter keys in watch face settings via Garmin Connect app
